## Technical Implementation

### Technology Stack

| Component | Technology | Rationale |
|-----------|------------|-----------|
| **Frontend** | Streamlit | Rapid development, Python-native, easy deployment |
| **Backend** | Python + FastAPI | High performance, async support, auto-documentation |
| **Database** | PostgreSQL | Robust, scalable, JSON support for flexibility |
| **Cache** | Redis | Fast sessions, rate limiting |
| **Hosting (POC)** | Railway | Free tier, easy deployment, includes PostgreSQL |
| **Data Sources** | GA4 API, GSC API | Direct from Google, most reliable data |

### Data Flow

```
Google Analytics 4 ──┐
                     ├──▶ ETL Pipeline ──▶ PostgreSQL ──▶ Dashboard
Google Search Console┘    (Daily Sync)      (Storage)      (Visualization)
```

### Security

- TLS 1.3 encryption for all connections
- JWT-based authentication with 30-minute session timeout
- Row-level security ensures clients see only their data
- OAuth 2.0 for Google API connections
- AES-256 encryption at rest
